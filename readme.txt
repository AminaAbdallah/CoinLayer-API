The coinLayer-API is a set of functions,a package ( module) that uses the coinLayer, it is implemented by Python for a training project.
The project has three main functions included  in a flolder named coinLayerAPI: Currency Conversion,Fetch Historical Data, 
and Fetch Data in a specific timeframe.
outside of the folder CoinlayerAPI , we find the file test.py in python, which contains the call of these fuctions to test them , which
